# Farming Guide — Coins, XP, and Materials

> **Applies to EA v0.1.1d / Updated: July 11, 2026**

Whether you need gold for vendor recipes, XP for Luminary points, or rare materials for endgame gear, this guide covers the most efficient farming methods in Witchspire. Each route is optimized for time-to-reward ratio, with familiar recommendations to maximize yield.

---

## Quick Coin Farming

### Best Overall: Shuddersands Relic Run

| Stat | Value |
|------|-------|
| Location | Lost Oasis (Shuddersands) |
| Coins per Hour | ~3,000-4,000 |
| Requirements | Level 24+, Steel tools, Heat Resist potions |
| Time per Run | 15-20 min |

**Route:**
1. Start at Valas Oasis (stock up on Heat Resist potions, repair gear)
2. Travel east to Lost Oasis
3. Loot all buildings for **Ancient Relics** (5-7 per run, 150 gold each)
4. Kill Dust Golems for Ancient Relic rare drops and Stone Blocks
5. Check the buried chest (requires Shovel) for extra gold
6. Return to Valas Oasis and sell everything to Nadia

**Total per run**: ~7 Ancient Relics (1,050 gold) + enemy drops (300-500 gold) + chest gold (500-800) = ~2,000 gold per 15 min.

### Best Early Game: Vyrs Honey + Nectar Loop

| Stat | Value |
|------|-------|
| Location | Vyrs Fields (starting region) |
| Coins per Hour | ~800-1,000 |
| Requirements | None |
| Time per Run | 5-7 min |

**Route:**
1. Start at Hearth
2. Run through Vyrs Fields collecting **Nectar** from yellow flowers (20-25 per run)
3. Sell Nectar to Mira the Vendor at Vyrs Crossroads (25 gold each)
4. Also collect raw Wood and Stone — sell excess to the traveling merchant

**Why it works**: Nectar respawns quickly (3-5 min) and vendors buy it at a fair price. Perfect for brand new characters.

### Best Mid-Game: Timberline Crystal Shard Run

| Stat | Value |
|------|-------|
| Location | Shard Caverns (Timberline Basin) |
| Coins per Hour | ~2,000-2,500 |
| Requirements | Level 12+, Iron+ tools |
| Time per Run | 10-12 min |

**Route:**
1. Enter Shard Caverns (east Timberline Basin, behind waterfall)
2. Head directly to **Crystal Vein Junction** (follow the crystal glow)
3. Mine all Crystal Shard nodes (20-30 per run)
4. Kill Crystal Crawlers for Chitin (sells for 40 gold each)
5. Check the two hidden chests inside the caverns
6. Exit and sell at Merchant Reyes (Basin Crossroads)

**Crystal Shard vendor price**: 50 gold each. A full run yields 1,000-1,500 gold from crystals alone.

## XP / Luminary Point Farming

### Best XP Farm: Shuddersands Nightveil Cave

| Stat | Value |
|------|-------|
| Location | Nightveil Cave (Shuddersands, west edge) |
| XP per Hour | ~8,000-10,000 |
| Requirements | Level 26+, Steel gear, access at night only |
| Time per Clear | 20-25 min |

**Strategy:**
1. Arrive at the cave entrance at 8 PM game-time
2. Clear all enemies (Void Crawlers + Nightveil Witches = high XP each)
3. Each enemy gives 150-250 XP
4. Optional boss (Nightveil Witch Matron) gives 1,500 XP + Veil Shard
5. Leave and rest at Valas Oasis inn — repeat after night cycle

**Pro tip**: Use XP-boosting food (Crystal Bud Stew — +15% XP for 30 min, crafted at Garden Plot + Furnace).

### Best Early XP: Vyrs Boar Hunting

| Stat | Value |
|------|-------|
| Location | Vyrs Meadow (north of Hearth) |
| XP per Hour | ~2,000-3,000 |
| Requirements | None |
| Time per Run | Continuous |

**Method:**
1. Kill Thornback Boars in Vyrs Meadow (3-5 per spawn cycle)
2. Each boar gives ~80 XP
3. Gather Raw Meat and Leather — craft into food or sell
4. Boars respawn in ~2 minutes — stay in the area and loop

### Best Mid XP: Timberline Spectral Witch Circuit

| Stat | Value |
|------|-------|
| Location | Al'uunian Tower (Timberline Basin) |
| XP per Hour | ~5,000-6,000 |
| Requirements | Level 14+, Iron+ gear |
| Time per Run | 15-20 min |

**Route:**
1. Enter Al'uunian Tower from ground floor
2. Clear Spectral Witches on ground floor (Lv 10-12, ~120 XP each)
3. Climb to upper floors — more witches + Crystal Golems (~180 XP each)
4. Optional: Kill Corrupted Crystal Golem (mid-boss, 2,000 XP)
5. Exit and repeat

## Material Farming Routes

### By Region

| Region | Best Materials | Best Location | Route Duration |
|--------|---------------|---------------|----------------|
| **Vyrs** | Wood, Stone, Iron Ore, Fiber, Nectar | Vyrs Meadow → South Cave → Fields loop | 15 min |
| **Timberline** | Crystal Shards, Coal, Moonleaf, Ethereal Dust | Shard Caverns + Al'uunian Tower loop | 20 min |
| **Mira Isles** | Pearl Shards, Ember Ore, Tropical Fruit, Ancient Wood | Soley Cave + Volcano Isle loop | 25 min |
| **Shuddersands** | Shimmer Sand, Astral Essence, Ancient Relics, Ethereal Dust | Glimmering Dunes + Lost Oasis loop | 30 min |

### Best Crystal Shard Farm
- **Location**: Shard Caverns (Timberline Basin) — Crystal Vein Junction
- **Yield**: 20-30 per 5 min
- **Best Familiar**: Crystal Sprite (increases crystal node yield by 15% passively)

### Best Ethereal Dust Farm
- **Location**: Nightveil Cave (Shuddersands) or Al'uunian Tower (Timberline)
- **Yield**: 4-6 per cave clear
- **Best Familiar**: Void Pup (increases ethereal enemy drops by 20%)

### Best Coal Farm
- **Location**: Shard Caverns (Timberline Basin) — dark stone deposits
- **Yield**: 8-12 per run
- **Alternative**: Buy from Merchant Reyes (Basin Crossroads) — 80 gold each

## Farming Tips

- **Use food buffs**: Thornberry Pie (+20% damage for faster kills) or Crystal Bud Stew (+15% XP)
- **Upgrade gathering tools first**: An Iron Pickaxe mines faster than a Stone Pickaxe. Upgrade priority: Pickaxe > Axe > Weapon
- **Empty your inventory before a farming run**: Keep only potions and a recall scroll
- **Set a Hearth flag at your farming location**: Quick teleport to vendor when full
- **Farm at night in Shuddersands**: 60% slower heat build-up = less downtime
- **Salvage gear from enemies**: Don't leave it on the ground — salvage at Workbench for partial materials
- **Multi-box with friends**: In co-op, have each player farm a different resource type, then share
