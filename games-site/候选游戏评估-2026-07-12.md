## 1. SpiritVale — 🥇 首选

| 维度 | 评估 |
|------|------|
| **游戏** | 类Ragnarok Online的MMORPG，F2P |
| **系统** | 7基础职业 + 8进阶职业、卡片系统、制作、Boss召唤、玩家经济 |
| **内容量** | 150+ 篇（职业×15 + 装备×438 + 卡片×227 + 地图×35 + Boss×20） |
| **上线** | Steam EA已上线，Demo可玩 |
| **Wiki** | ❌ 无独立Wiki。零星攻略分散在 lagofast/grindnstrat/gamerblurb |
| **机会** | F2P → 玩家基数大；MMORPG → 攻略需求极强；无Wiki → 蓝海 |
| **风险** | EA阶段，正式版可能 wipe 进度 |

## 2. Skills & Raids — 🥈 独特品类

| 维度 | 评估 |
|------|------|
| **游戏** | 队伍制提取RPG（BG3 × Tarkov），单人开发 |
| **系统** | 队伍战斗、技能混合Build、5级稀有度、制作、提取循环 |
| **内容量** | 80+ 篇（Build×30 + 技能×20 + 敌人×15 + 地图×10 + 制作×10） |
| **上线** | 7月，Open Beta 可玩 |
| **Wiki** | ❌ 完全零覆盖，连一篇深度攻略都没有 |
| **机会** | 提取类正在爆发（Mistfall Hunter/The Mound都是提取）+ 独立开发者不会做官方Wiki |
| **风险** | 单人开发，游戏可能体量有限；最坏情况是游戏暴死 |
