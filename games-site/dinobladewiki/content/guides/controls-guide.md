# Dinoblade Controls Guide — All Platforms and Settings

Dinoblade is currently available on PC via Steam (Windows). This guide covers the default control bindings for both keyboard-and-mouse and controller setups, along with system requirements and recommended settings for smooth performance.

---

## Keyboard and Mouse Controls

All keys are rebindable from the in-game settings menu. The defaults are optimized for a standard QWERTY layout.

### Combat Controls

| Action | Default Key | Notes |
|--------|-------------|-------|
| **Light Attack** | Left Mouse Click | Can be chained up to 4 times. Third hit in the chain is a guaranteed stagger on most enemies. |
| **Heavy Attack** | Hold Left Click / Right Click | Charge increases damage up to 2x. Fully charged attacks break guard and cause heavy stagger. |
| **Roll / Dodge** | Space | Grants invincibility frames (i-frames) at the start of the roll. 11 i-frames at 60fps. |
| **Parry** | F | 5 active frames. Must be timed to land during an enemy attack's active hit window. |
| **Lock On** | Middle Mouse Click | Toggle targeting. Disengages if the target moves out of range. |
| **Special Skill** | R | Each weapon has a unique special skill. Uses Spirit points. |
| **Boss Soul Summon** | T | Requires an equipped boss soul. Consumes the soul on use (single powerful attack). |

### Movement and Interaction

| Action | Default Key | Notes |
|--------|-------------|-------|
| **Move** | W, A, S, D | Standard WASD movement relative to camera. |
| **Sprint** | Left Shift (hold) | Drains stamina while active. Cannot sprint with empty stamina. |
| **Walk** | Ctrl (hold) | Silent movement. Useful for sneaking up on enemies. |
| **Interact** | E | Pick up items, open doors/chests, talk to NPCs, activate Roosts. |
| **Use Item** | Q | Uses the currently selected consumable (Estus flask by default). |
| **Cycle Items** | Mouse Wheel Up/Down | Scroll through your item bar. |
| **Toggle Weapon Stance** | 1 / 2 | Switch between one-handed and two-handed grip. Two-handed increases damage by 15% but disables off-hand parry. |

### Camera Controls

| Action | Default Key | Notes |
|--------|-------------|-------|
| **Camera Look** | Mouse movement | Sensitivity is highly configurable (see settings below). |
| **Reset Camera** | Middle Mouse Click (hold + release) | Snaps camera behind your character. |
| **Zoom In/Out** | Mouse Wheel (with weapon sheathed) | Adjusts third-person camera distance. |

## Controller Controls

Dinoblade fully supports Xbox and PlayStation controllers. Detection is automatic — the UI switches button prompts to match your controller.

### Combat Controls

| Action | Xbox | PlayStation |
|--------|------|-------------|
| **Light Attack** | RB | R1 |
| **Heavy Attack** | RT | R2 |
| **Roll / Dodge** | A | Cross |
| **Parry** | LB | L1 |
| **Special Skill** | B | Circle |
| **Boss Soul Summon** | B (hold) | Circle (hold) |
| **Lock On** | Right Stick Click | R3 |

### Movement and Interaction

| Action | Xbox | PlayStation |
|--------|------|-------------|
| **Move** | Left Stick | Left Stick |
| **Sprint** | Left Stick Click | L3 |
| **Interact** | Y | Triangle |
| **Use Item** | X | Square |
| **Cycle Items (Next)** | D-Pad Right | D-Pad Right |
| **Cycle Items (Previous)** | D-Pad Left | D-Pad Left |
| **Toggle Weapon Stance** | D-Pad Up | D-Pad Up |
| **Camera** | Right Stick | Right Stick |
| **Reset Camera** | Right Stick Click (hold + release) | R3 (hold + release) |

### Controller Vibration Settings

| Setting | Default | Recommended |
|---------|---------|-------------|
| **Vibration Intensity** | 100% | 70% — helps feedback without fatigue over long sessions |
| **Vibration on Parry** | On | On — critical tactile cue for parry timing |
| **Vibration on Hit** | On | On |
| **Vibration on Death** | On | Off — personal preference |

## Recommended Camera Settings

| Setting | Recommended Value | Notes |
|---------|------------------|-------|
| **Camera Sensitivity X** | 5 | Adjust based on your mouse/controller preference |
| **Camera Sensitivity Y** | 4 | Slightly lower vertical sensitivity helps with precision aiming of attacks |
| **Camera Distance** | 8 | Max distance gives the widest field of view in boss fights |
| **Camera Speed (Controller)** | 7 | Fast enough to track Kira and Kasei's movements |
| **Auto-Center Camera** | Off | Manual control gives better positioning for parries |
| **Motion Blur** | Off | Disabling motion blur makes parry windows significantly easier to see |
| **Head Bob** | Off | Reduces motion sickness risk during extended sessions |
| **Field of View** | 90 | Standard FPS FoV; wider than the default 80 |

## System Requirements and Performance Settings

### Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| **OS** | Windows 10 64-bit | Windows 11 64-bit |
| **CPU** | Intel i5-8400 / AMD Ryzen 5 2600 | Intel i7-10700 / AMD Ryzen 7 3700X |
| **RAM** | 8 GB | 16 GB |
| **GPU** | NVIDIA GTX 1060 6GB / AMD RX 580 | NVIDIA RTX 3060 / AMD RX 6700 XT |
| **Storage** | 40 GB SSD | 40 GB SSD |
| **DirectX** | Version 11 | Version 12 |

### Performance Presets

| Preset | Target Spec | Expected Performance |
|--------|-------------|---------------------|
| **Low** | Below minimum | 30-45 fps at 1080p. Use if you have a GTX 960-equivalent GPU. |
| **Medium** | Minimum specs | 50-60 fps at 1080p. Best balance of visuals and performance. |
| **High** | Recommended specs | 60 fps at 1080p/1440p. Full visual fidelity. |
| **Ultra** | Above recommended | 60+ fps at 4K. Requires RTX 3080 or better. |

### Key Settings for 60fps

If you are targeting 60fps on minimum-spec hardware, prioritize these settings:

| Setting | Value | Performance Impact |
|---------|-------|-------------------|
| **Resolution** | 1920 x 1080 | Baseline |
| **Shadow Quality** | Low | Large GPU savings |
| **Texture Quality** | Medium | Moderate GPU savings |
| **Ambient Occlusion** | Off | Significant GPU savings |
| **Anti-Aliasing** | TAA (Low) | Keeps edges clean with minimal cost |
| **Post-Processing** | Low | Disables depth of field and bloom |
| **View Distance** | Medium | Balances CPU and GPU load |
| **VSync** | On | Prevents screen tearing |

### Advanced Settings

| Setting | Options | Recommendation |
|---------|---------|---------------|
| **Render Scaling** | 50%-200% | 100% for standard displays; 150% for sharpening on high-DPI monitors |
| **FPS Cap** | 30/60/120/Unlimited | 60 for consistency; 120+ only on high-end hardware |
| **Upscaling** | DLSS (RTX), FSR (all GPUs) | FSR Quality mode works on any GPU and provides a 20-30% fps boost |
| **HDR** | On/Off | Enable if your display supports it; notably improves Crystal Caverns lighting |

## Accessibility Options

| Setting | Options | Notes |
|---------|---------|-------|
| **Subtitle Size** | Small / Medium / Large | Helps readability at a distance |
| **Colorblind Mode** | Protanopia / Deuteranopia / Tritanopia | Changes health bar, enemy lock-on, and UI colors |
| **Auto-Target** | On / Off | Off by default. On auto-locks nearest enemy when attacking. |
| **Combat Text Size** | Small / Medium / Large | Damage numbers and status effect text |
| **Screen Shake** | 0% - 100% | Reducing this helps players prone to motion sickness |
| **Quick Time Events** | Hold / Tap / Off | Some boss interactions use QTEs; "Hold" mode reduces dexterity requirement |
