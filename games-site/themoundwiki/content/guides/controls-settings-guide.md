## Recommended Graphics Settings

### For Maximum Visibility (Competitive / Survival Focus)

| Setting | Recommendation | Why |
|---------|---------------|-----|
| Brightness | +1 to +2 notches above default | Dark zones are very dark; extra brightness reveals hidden threats |
| Field of View | 90-100 | Wider FOV helps spot flanking enemies |
| Texture Quality | High | Enemy visibility in vegetation depends on texture clarity |
| Shadow Quality | Medium | Shadows are important for spotting patrols, but Ultra costs frames |
| Ambient Occlusion | On | Helps distinguish real objects from hallucinations (better depth) |
| Anti-Aliasing | TAA or DLSS (Quality) | Reduces visual noise that can mimic hallucination effects |
| Volumetric Fog | Low | Fog is atmospheric but obscures threats; lower = clearer vision |
| Motion Blur | Off | Motion blur disorients during combat |
| V-Sync | Off (use frame limiter instead) | Reduces input lag |
| HDR | On (if supported) | Improves dark-area visibility |

### For Maximum Performance

| Setting | Recommendation |
|---------|---------------|
| Resolution | Native monitor resolution |
| DLSS / FSR | Performance mode |
| Texture Quality | Medium |
| Shadow Quality | Low |
| Volumetric Fog | Off |
| Post Processing | Low |
| Anti-Aliasing | Low |

## Recommended Audio Settings

| Setting | Recommendation | Why |
|---------|---------------|-----|
| Master Volume | 70% | Headroom for loud combat sounds |
| SFX Volume | 100% | Footsteps, enemy sounds, environmental cues |
| Voice Chat Volume | 80% | Clear communication without drowning out game audio |
| Music Volume | 50% | Atmosphere without masking enemy audio |
| Spatial Audio | On (Windows Sonic / Tempest 3D / Dolby Atmos) | Essential for direction-based threat detection |
| Voice Chat Mode | Push to Talk (PC) / Open Mic (Console) | Avoids accidental broadcast on PC; convenience on console |
| Headphone Mode | On | Optimized audio mix for headphones |

**Headphones are strongly recommended.** Spatial audio is critical for detecting enemy direction and distance.

## Common Technical Issues and Fixes

### Issue: Game crashes on launch
**Fix:** Verify game files (Steam: Properties > Local Files > Verify). Update GPU drivers. Disable overlays (Discord, NVIDIA).

### Issue: Low FPS / Stuttering
**Fix:** Lower Volumetric Fog and Shadow Quality. Enable DLSS or FSR. Close background applications. Check CPU/GPU temperatures.

### Issue: Voice chat not working
**Fix:** Check voice chat is enabled in audio settings. Verify microphone permissions in OS settings. Test push-to-talk keybind. Check NAT type (cross-platform voice requires open NAT).

### Issue: Spatial audio not working
**Fix:** Ensure spatial audio is enabled in Windows sound settings. Set game to Headphone mode. Stereo vs surround — the game uses stereo spatial, do not force 7.1.

### Issue: Controller not detected
**Fix:** Disconnect and reconnect controller. Use USB cable instead of Bluetooth. Launch the game with controller already connected. Check Steam Input settings.

### Issue: Screen too dark
**Fix:** Increase in-game brightness. Disable HDR if display does not properly support it. Check GPU control panel brightness settings.
