## Stats & Attributes Overview

Echoes of Aincrad uses seven core attributes. Every time you level up, you earn **Growth Points** to spend at any Inn. Stat allocation dramatically impacts damage, available Sword Skills, and combat effectiveness.

| Stat | Function |
|------|----------|
| Strength (STR) | Physical damage, heavy weapon scaling |
| Dexterity (DEX) | Crit rate, light weapon damage |
| Agility (AGI) | Attack speed, dodge recovery |
| Intelligence (INT) | Skill damage, magical Sword Skills |
| Vitality (VIT) | Maximum HP, physical defense |
| Endurance (END) | Maximum SP, stamina recovery |
| Mind (MIND) | Support skill effectiveness |

## Stat Details & Breakpoints

### Strength (STR)

Primary scaling for Greatswords and Two-Handed Axes.

| STR | Damage Bonus | Breakpoint Bonus |
|-----|-------------|-----------------|
| 1–10 | +3% per point | — |
| 11–20 | +2% per point | Unlocks Greatsword heavy slam |
| 21–30 | +1.5% per point | Guard Break Up passive |
| 31–40 | +1% per point | Unlocks Two-Handed Axe spin |
| 41+ | +0.5% per point | Diminishing returns |

**Soft cap**: 20 | **Hard cap**: 40

### Dexterity (DEX)

Primary scaling for Rapiers and Daggers. Increases crit rate.

| DEX | Crit Rate | Breakpoint Bonus |
|-----|----------|-----------------|
| 1–10 | +1.5% per point | — |
| 11–20 | +1.0% per point | Parry window +0.1s |
| 21–30 | +0.8% per point | Unlocks Dagger backstab |
| 31–40 | +0.5% per point | Rapier thrust combo extended |
| 41+ | +0.3% per point | Diminishing returns |

**Soft cap**: 20 | **Hard cap**: 40

### Agility (AGI)

Improves attack speed and dodge recovery. Often undervalued.

- **1–10**: +2% attack speed per point
- **11–20**: +1.5% per point, dodge recovery -0.1s
- **21–30**: +1% per point, dodge recovery -0.1s
- **31+**: +0.5% per point

### Intelligence (INT)

Primary stat for Mage builds (Wand/Staff). Scales magical Sword Skills.

| INT | Skill Damage | Breakpoint Bonus |
|-----|-------------|-----------------|
| 1–10 | +4% per point | — |
| 11–20 | +3% per point | Unlocks AoE spell |
| 21–30 | +2% per point | Mana recovery increased |
| 31–40 | +1.5% per point | Unlocks ultimate spell |
| 41+ | +1% per point | Diminishing returns |

**Soft cap**: 20 (hybrid) | **Hard cap**: 40 (pure mage)

### Vitality (VIT)

HP and physical defense. Every build needs some.

- **1–10**: +50 HP / +1 def per point
- **11–20**: +40 HP / +1 def per point
- **21–30**: +30 HP / +1 def per point
- **31+**: +20 HP / +1 def per point

**Minimum**: 10 VIT | **Tank**: 25+ VIT

### Endurance (END)

SP pool and recovery. The most important secondary stat — you cannot act without SP.

| END | SP Bonus | Recovery Bonus |
|-----|---------|---------------|
| 1–10 | +15 SP per point | +3% per point |
| 11–20 | +12 SP per point | +2% per point |
| 21–30 | +10 SP per point | +1.5% per point |
| 31+ | +8 SP per point | +1% per point |

**Minimum**: 15 END for all builds.

### Mind (MIND)

Support skill effectiveness and buff duration.

- **1–10**: +5% support effectiveness per point
- **11–20**: +3% buff duration per point
- **21+**: +2% support effectiveness per point

Best for support-focused playstyles and extensive partner skill use.

## Frequently Asked Questions

**What is the best stat to max first?** END (15 minimum), then your primary damage stat based on weapon.

**Is INT useless for melee?** Yes. There is no hybrid melee-mage scaling.

**Does AGI stack with attack speed EX-Mods?** Yes, creating the fastest possible attack animations.

**Should I invest in MIND for solo play?** Only if you use partner support skills frequently.

**Glass cannon viable?** Yes for experienced players, but beginners should invest in VIT and END as secondary priorities.
