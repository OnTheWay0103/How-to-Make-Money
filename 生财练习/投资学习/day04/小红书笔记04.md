# 一张图搞懂基金分类，再也不怕看不懂了🧠

姐妹们，基金真的没那么复杂
一张思维导图帮你理清楚👇

---

【第一层：按投资对象分】

📌 **货币基金**
→ 投资短期货币工具
→ 代表：余额宝、零钱通
→ 风险：⭐（极低）
→ 收益：1-2%

📌 **债券基金**
→ 投资各类债券
→ 又分：纯债/一级债/二级债/可转债
→ 风险：⭐⭐
→ 收益：3-5%

📌 **股票基金**
→ 80%以上资产投股票
→ 波动最大，长期收益也最高
→ 风险：⭐⭐⭐⭐
→ 收益：浮动大，长期8-10%

📌 **混合基金**
→ 股票+债券灵活配置
→ 股市好时多配股票，不好时多配债券
→ 风险：⭐⭐⭐
→ 收益：介于股基和债基之间

---

【第二层：按管理方式分】

🤖 **被动基金（指数基金）**
→ 不挑股票，复制指数
→ 费率低（0.15%/年）
→ 长期85-90%的主动基金跑不赢它
→ 普通人首选‼️

👨‍💼 **主动基金**
→ 基金经理选股择时
→ 费率高（1.5%/年）
→ 依赖基金经理能力
→ 长期跑赢指数的不到15%

---

【第三层：按募集方式分】

🏢 **公募基金**
→ 1元起投，谁都能买
→ 每天公布净值
→ 监管严格
→ 普通人买的就是这类

🏦 **私募基金**
→ 100万起投
→ 信息不公开
→ 门槛高、风险大
→ 不适合小白

---

【总结一句话】

✅ 新手买什么？
**公募 + 被动 + 指数基金 = 最适合普通人的组合**

✅ 从哪里开始？
**沪深300指数基金，费率最低的那只**

✅ 千万别碰什么？
**别一上来就买主动基金（费率高）**
**别碰私募基金（门槛高风险大）**
