# 宽基 vs 行业，新手先买哪个📊

宽基和行业我都买了
有赚有亏，今天学完终于理清楚了👇

---

**📌 宽基指数（主食）**
跨行业选股，风险分散
→ 沪深300、中证500、标普500
→ 波动小，拿得住
→ 适合：所有人，尤其是新手
→ 建议占比：70-80%

**📌 行业指数（调味料）**
只买一个行业，集中押注
→ 白酒、医疗、新能源、半导体
→ 涨起来爽，跌起来也痛
→ 适合：有经验、能扛跌的人
→ 建议占比：20-30%
