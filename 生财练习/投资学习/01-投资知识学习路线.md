## 第1周：钱的基础认知

### Day 1：通货膨胀——你的钱在悄悄变少

**学（30分钟）**
- 什么是通货膨胀：物价上涨，钱的购买力下降
- 中国过去20年平均通胀率约2-3%（官方CPI数据）
- 搜一篇通俗文章：《通货膨胀是如何偷走你的钱的》
- 关键概念：名义收益 vs 实际收益（实际收益 = 名义收益 - 通胀率）

**练（30分钟）**
- 打开手机计算器，算以下三笔账：
  1. 10万放银行活期（年化0.3%），扣掉3%通胀，10年后实际购买力剩多少
  2. 10万放余额宝（年化1.5%），扣掉3%通胀，10年后实际购买力剩多少
  3. 10万定投指数基金（假设年化8%），扣掉3%通胀，10年后实际购买力剩多少
- 把计算结果记下来，贴在你能看到的地方

**产出素材**
- 小红书：三行数字对比，标题《放银行10年后还剩多少？算完不淡定了》

### Day 2：复利——世界第八大奇迹

**学（30分钟）**
- 复利公式：终值 = 本金 × (1 + 收益率)^年数
- 复利 vs 单利的区别（用表格对比）
- 72法则：72 ÷ 年化收益率 = 本金翻倍需要的年数
- 复利三要素：本金、收益率、时间（时间是最重要的变量）

**练（30分钟）**
- 打开复利计算器网站或用Excel，算以下场景：
  1. 10万本金，年化8%，10年/20年/30年后各是多少
  2. 10万本金 + 每月追加1万，年化8%，10年/20年/30年后各是多少
  3. 同样条件，年化5% vs 8% vs 10%，30年后差距有多大
- 把最震惊的一组数字圈出来，这就是你坚持定投的动力

**产出素材**
- 小红书：一张复利计算对比表，标题《同样的钱，放不同地方30年差了多少》

### Day 6：认识风险——你能承受亏多少钱

**学（30分钟）**
- 投资中的风险不是"亏钱"，是"不确定"
- 波动 ≠ 永久性亏损：跌了会回来 vs 跌了永远回不来
- 最大回撤：一个投资品历史上最多跌过多少
- 风险与收益的关系：高收益一定伴随高风险，没有例外
- 常见骗局识别：任何承诺"稳赚不赔""保本高收益"的都是骗局

**练（30分钟）**
- 做一份个人风险承受能力自测，诚实写下来：
  1. 10万本金，跌到8万（亏20%），你的反应是？（A.加仓 B.不动 C.卖出 D.睡不着）
  2. 跌到6万（亏40%），你的反应是？
  3. 你能接受的最大年度亏损是多少？（5%/10%/20%/30%）
  4. 这笔钱3-5年内会不会急用？
  5. 根据你的答案，判断你适合偏保守还是偏进取的配置

## 第2周：基金入门

### Day 8：基金到底是什么

**学（30分钟）**
- 基金的法律结构：投资人→基金公司→托管银行→投资标的
- 净值（NAV）是什么：一份基金值多少钱
- 基金是怎么赚钱的：净值增长 + 分红
- 基金费用全解：管理费（每年）、托管费（每年）、申购费（买入时）、赎回费（卖出时）
- 关键认知：费用是投资回报的隐形杀手，费率差0.5%，30年差出一辆车

**练（30分钟）**
- 打开天天基金，挑一只知名基金（如易方达沪深300ETF联接）：
  1. 找出管理费率、托管费率、申购费率、赎回费率
  2. 算一笔账：投1万，持有一年，各种费用一共多少
  3. 对比另一只同类基金的费用，看有没有差异
  4. 记录下来，养成"买之前先看费率"的习惯

### Day 9：主动基金 vs 指数基金

**学（30分钟）**
- 主动基金：基金经理选股择时，目标是跑赢市场
- 指数基金：被动跟踪指数，目标是复制市场收益
- 关键数据：长期来看，85-90%的主动基金跑不赢指数（SPIVA报告）
- 原因：主动基金管理费高 + 人很难持续战胜市场
- 结论：普通人首选指数基金，费用低、逻辑透明、不依赖基金经理

**练（30分钟）**
- 实操对比：
  1. 找一只沪深300指数基金（如易方达沪深300ETF联接）
  2. 找一只主动管理的股票基金（规模大、成立时间长的那种）
  3. 对比：近1年/3年/5年收益率 + 管理费率
  4. 算一笔账：假设各投10万，持有5年，在扣费后分别赚多少
  5. 记录你的发现

**产出素材**
- 公众号：《我为什么劝新手别买主动基金》，附对比数据

### Day 10：什么是指数——沪深300、中证500到底是什么

**学（30分钟）**
- 指数是什么：按照一定规则选出的一篮子股票，反映某个市场或行业的整体走势
- A股核心指数：
  - 沪深300：A股市值最大、流动性最好的300家公司（大盘蓝筹）
  - 中证500：排在沪深300之后的中盘股
  - 中证1000：小盘股
  - 创业板指：创业板中的100只代表
  - 科创50：科创板50只龙头
- 为什么指数长期一定涨：经济在增长，好公司利润在增长，指数成分股定期调换（劣汰优留）

**练（30分钟）**
- 打开天天基金或雪球：
  1. 搜"沪深300"，看当前点位、近1年/3年/5年走势
  2. 搜"中证500"，做同样操作
  3. 对比两只指数的历史走势，谁波动更大？谁长期涨得多？
  4. 看沪深300的前十大持仓股（如贵州茅台、宁德时代等），你是否认识这些公司
  5. 想想：买沪深300指数基金 = 一次性买了这300家公司的一部分

**产出素材**

### Day 11：宽基指数 vs 行业指数——新手先投哪个

**学（30分钟）**
- 宽基指数：覆盖多个行业，分散风险（沪深300、中证500）
- 行业指数：只覆盖某一个行业（消费、医药、新能源、芯片等）
- 行业指数的风险高在哪：一个行业可能长期不涨甚至下跌（参考教培行业政策冲击）
- 国外核心指数：标普500（美国500家最大公司）、纳斯达克100（美国科技巨头）
- 新手原则：先宽基后行业，先国内后海外

**练（30分钟）**
- 实操对比行业指数：
  1. 搜"中证白酒"指数，看近3年走势
  2. 搜"中证医疗"指数，看近3年走势
  3. 搜"中证新能源"指数，看近3年走势
  4. 把这三个行业指数和沪深300放一起对比：哪个波动最大？如果你是新手买了波动最大的那个，中间亏了30%你拿得住吗？
  5. 写下结论：为什么新手从宽基开始

### Day 12：ETF、LOF、ETF联接——这些字母到底什么意思

**学（30分钟）**
- ETF（交易所交易基金）：像股票一样买卖，实时交易
- 场外基金：通过基金公司或代销平台申购，按收盘净值成交
- ETF联接：给没有证券账户的人买ETF的渠道，本质是场外基金
- LOF：既可以在场内交易也可以在场外申赎
- 基金名字里的A/C：A类收申购费、免销售服务费 → 适合长期持有（1年以上）；C类免申购费、收销售服务费 → 适合短期持有（1年内）
- 增强指数基金：在跟踪指数基础上做点主动操作，目标是比指数多赚一点（也可能多亏）

**练（30分钟）**
- 在天天基金上实操：
  1. 搜"沪深300ETF"，看场内ETF和场外联接基金的区别
  2. 找同一基金的A类和C类，对比费率和买入费率
  3. 算一笔账：投1万，持有1年 vs 3年，选A类还是C类更划算
  4. 搜一只"增强指数基金"，看它近3年有没有跑赢对应的普通指数基金

**产出素材**

### Day 13：怎么挑一只好的指数基金

**学（30分钟）**
- 挑选指数基金的5个关键指标：
  1. 基金规模：至少1亿以上（太小可能清盘）
  2. 跟踪误差：越小越好，说明基金紧跟指数
  3. 费率：管理费+托管费，越低越好
  4. 成立时间：至少3年以上，有历史可查
  5. 基金公司：头部公司风控更强（易方达、华夏、南方、富国等）
- 同一指数的不同基金，长期收益差距主要来自费率差异

**练（30分钟）**
- 做一遍完整的选基金流程：
  1. 搜索跟踪"沪深300"的所有指数基金
  2. 筛出规模>1亿、成立>3年的
  3. 按费率从低到高排列
  4. 挑出3只候选基金
  5. 在一张表里对比它们的：规模、费率、跟踪误差、近3年收益、基金公司
  6. 选出你认为最好的一只，写下选择理由

### Day 16：定投策略——为什么定期定额是最优解

**学（30分钟）**
- 定投核心逻辑：不是预测市场，是放弃择时
- 微笑曲线：越跌越买、摊低成本、等市场回升时赚钱
- 用数字说话：假设基金先跌后涨，单笔买入 vs 定投的收益对比
- 定投的反直觉真相：开始定投后短期内下跌反而是好事，因为你在便宜的时候买得更多
- 定投的局限性：长期向上的资产才适合定投

**练（30分钟）**
- 用定投计算器实操（天天基金上就有，或网上搜"定投收益计算器"）：
  1. 选沪深300指数，设定每月定投1000元
  2. 分别计算：过去3年、5年、10年的定投收益
  3. 对比一次性买入和定投的收益差
  4. 特别注意：中间最大回撤有多少（你定投时可能也要经历）
  5. 写下感悟：如果中间亏了20%，你能坚持继续投吗

### Day 17：第一笔定投——今天就开始

**学（30分钟）**
- 定投参数设置：
  - 投多少：建议从月入闲钱的10-20%开始（比如每月1000-2000）
  - 频率：周投或月投，长期差别不大，月投省心
  - 日期：月投选发工资后2-3天，比如每月10号
  - 扣款方式：从银行卡自动扣款
- 定投纪律：设置好之后，不要每个月手动调整金额和日期

**练（30分钟）**
- 设置你的第一笔定投计划：
  1. 在支付宝或天天基金找到你选的指数基金
  2. 点击"定投"
  3. 设置：金额（建议500-2000元）、周期（月投）、日期（如每月10号）、付款方式
  4. 完成设置，截图留念

### Day 18：设置完之后做什么——什么也不做

**学（30分钟）**
- 定投之后最关键的原则：少看账户
- 为什么少看：每一次看都是在和人性斗争（赚了想卖、亏了想跑）

- 什么情况下应该关注：按月或按季定期复盘（固定日期），而不是因为涨了或跌了
- 投资最大的敌人不是市场，是自己的情绪

**练（30分钟）**

## 第4周：加深理解 + 建立体系

### Day 22：资产配置——不只投指数基金

**学（30分钟）**
- "四笔钱"框架：
  1. 活钱（3-6个月生活费）：流动性要高，放货基
  2. 稳钱（1-3年内要用的）：波动要低，放债基
  3. 长钱（3年以上不用的）：博取收益，放指数基金
  4. 保钱（医疗意外）：保障型保险
- 股债平衡原理：股票和债券通常不同涨同跌，搭配使用能降低整体波动
- 简单规则：留够活钱和稳钱，剩下的才能投指数基金

**练（30分钟）**
- 把你的每笔钱分到这4个桶里：
  1. 列出你当前的每笔资金
  2. 确认：活钱够不够（月支出×6）
  3. 确认：接下来1-3年有没有大额支出（买房/结婚等）
  4. 确定：真正可以长期不动用来投资的是多少
  5. 这笔"长钱"就是你的定投资金来源，算出来你每月最多能投多少

**产出素材**
- 小红书：《把你的钱分成这4笔再开始理财》，一张分区图讲清楚

### Day 23：设计你的投资组合

**学（30分钟）**
- 最简单的组合方案：宽基指数组合
  - 保守型：80%债基 + 20%指数基金
  - 平衡型：50%债基 + 50%指数基金
  - 进取型：20%债基 + 80%指数基金
- 100-年龄法则：指数基金占比 = 100 - 你的年龄
- 再平衡：每年固定时间（如1月1日）把偏离的比例调回来

**练（30分钟）**

### Day 24：估值入门——现在的市场是贵还是便宜

**学（30分钟）**
- PE（市盈率）= 股价 ÷ 每股盈利，低PE代表便宜，高PE代表贵
- PE分位点：当前PE在过去10年中处于什么位置
  - 分位点<30%：相对便宜
  - 分位点30-70%：合理
  - 分位点>70%：相对偏贵
- 指数基金的估值怎么看 → 看这个指数整体的PE分位点
- 一个重要认知：估值低不意味着马上会涨，估值高不意味着马上会跌，但拉长时间看，低估值买入的长期收益更好

**练（30分钟）**
- 实操查估值：
  1. 打开天天基金或雪球，搜"沪深300 PE"
  2. 看当前的PE值和PE分位点
  3. 再查中证500的PE分位点
  4. 对比：现在哪个指数更便宜？
  5. 记录：当前的PE分位点是多少，对应市场是贵还是便宜

**产出素材**
- 小红书：《现在适合入场吗？看一眼这个指标就知道了》，手把手教查PE分位点

### Day 25：行为金融学——为什么大多数人亏钱

**学（30分钟）**
- 投资中最常见的心理陷阱：
  1. 追涨杀跌：涨了觉得还能涨赶紧买、跌了怕继续跌赶紧卖（高买低卖）
  2. 损失厌恶：亏1万的痛苦 ≈ 赚2.5万的快乐
  3. 锚定效应：死盯着买入价，跌了不甘心卖，涨一点就赶紧跑
  4. 从众心理：周围的人都在买（或都在卖），你也跟着做
  5. 过度自信：觉得自己能预测市场
- 核心认知：投资赚钱 = 正确的认知 + 纪律执行，和智商关系不大

**练（30分钟）**
- 对照自检：
  1. 过去一周，你想过几次"要不要打开账户看看涨了还是跌了"
  2. 你最近刷到过几次"某股票/基金暴涨"的消息，有没有心动的感觉
  3. 写下来：如果明天你的定投账户跌了5%，你第一反应想做什么
  4. 再写：你实际应该做什么（根据Day18制定的纪律）
  5. 把这两个答案放在一起，思考：情绪和纪律之间有多大的差距

**产出素材**
- 公众号：《亏钱不是因为你不懂，是因为你是人》，讲行为金融学的5个陷阱

### Day 26：投资中的交易成本

**学（30分钟）**
- 基金持有成本全景：
  - 买入成本：申购费（通常打1折后0.15%左右）
  - 持有成本：管理费+托管费（ETF通常0.5%/年，主动基金1.5%/年）
  - 卖出成本：赎回费（持有<7天1.5%惩罚性，7天-1年0.5%，>1年通常0%）
- 交易成本对长期收益的侵蚀：1%的费率差，30年能吃掉30%+的收益
- A股交易费用：印花税（卖出时0.05%）、佣金、过户费

**练（30分钟）**
- 算一笔完整的费用账：
  1. 你在Day17设置的定投，每笔申购费是多少
  2. 你的基金每年管理费+托管费一共多少
  3. 假设你持有3年赎回，总费用占本金的百分比是多少
  4. 再算如果买的是一只主动基金（管理费1.5%），持有3年的总费用是多少
  5. 把两个数字放一起，理解为什么指数基金适合普通人

**产出素材**

### Day 27：建立你的投资规则书（IPS）

**学（30分钟）**
- 投资政策声明（IPS, Investment Policy Statement）：一份写给你自己未来的规则书
- 为什么需要IPS：市场跌了30%的时候，你是情绪主导的人，现在冷静的你应该提前帮那时候的自己做好决定
- IPS的核心内容：
  1. 投资目标：为了什么而投资（养老/教育/财务自由）
  2. 资产配置：股票%+债券%
  3. 定投规则：每月投多少、投什么
  4. 再平衡规则：每年什么时候调整
  5. 调仓触发条件：什么情况下才更改方案（一般只有人生大事才改）
  6. 操作禁区：绝对不做什么（如不追热点、不单押行业、不满仓一只基金）

**练（30分钟）**
- 写你的个人IPS，包含以下条款：
  1. 投资目标：_______________
  2. 资产配置比例：指数基金___% + 债券基金___%
  3. 定投方案：每月___日投___元到（基金名称）
  4. 再平衡：每年___月___日检查并调整
  5. 我绝对不做的3件事：
  6. 我每月只在___号看一次账户
- 把这份IPS打印出来或存到手机备忘录置顶
