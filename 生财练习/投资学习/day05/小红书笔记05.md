# 投资品种大盘点！看这一篇就够了📖

姐妹们，学了6天终于把市面上所有投资品种摸清了

一张总图帮你理清楚👇

---

🏦 **现金管理类**
→ 现金、货币基金（余额宝）
→ 管的是"随时要用的活钱"
→ 收益低但随取随用

📜 **固定收益类**
→ 国债、企业债、债券基金
→ 管的是"1-3年不用的钱"
→ 稳健增值，波动小

📈 **权益类**
→ 股票（A股/港股/美股）
→ 基金（主动/被动/指数）
→ 管的是"3年以上不用的钱"
→ 长期收益最高，波动也最大

🏠 **实物资产类**
→ 房产（逻辑已变，别乱买）
→ 黄金（不生息，别做短线）
→ 配置5-10%就够了

🛡️ **风险转移类**
→ 医疗/重疾/意外/寿险
→ 管的是"万一出事怎么办"
→ 先保障后理财
