# 买基金到底用哪个App？我选了微信📱

学了两周终于要下单了
结果卡在一个问题上——

**基金到底在哪儿买？？**

研究了一圈，分享给大家👇

---

**📱 主流买基金渠道对比**

**微信理财通 / 支付宝**
→ 申购费1折，最省钱
→ App内直接买，最方便

**天天基金**
→ 数据最全，适合深度研究
→ 但App功能太多，上手慢一点

**银行App（招行、工行等）**
→ 申购费不打折，最贵！
→ 同样1万块，银行收150，互联网平台收15

**基金公司官网**
→ 有些免申购费
→ 但只能买一家的，不方便

---

**我的选择：微信理财通 👇**

原因很简单：
我不怎么用支付宝
日常支付、转账、零钱都在微信里
不想为了买基金多装一个App

微信搜"理财通"就能进
基金种类够用，定投设置也方便
对我这种新手来说足够了✅

---

**⚠️ 但微信有个小缺点**

它筛选基金不太方便
不能按费率从低到高排序
也不能一键对比历史收益

所以我干了一件事——

---

**🤖 用AI给自己做了个小工具**

🔗 **fundfind.streamlit.app**

功能很简单：
搜指数 → 筛规模 → 按费率排序 → 推荐前三

沪深300、中证500、标普500……
常用的指数都内置了

最关键是——
**我不会写代码，全程用AI做的。**

把需求告诉AI → AI帮我写代码 → 部署到网上 → 现在谁都能用
