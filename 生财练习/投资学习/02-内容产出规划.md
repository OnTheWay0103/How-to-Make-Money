# 投资内容产出规划

> 目标：边学边产出，公众号做深度+陪伴，小红书做碎片+钩子，两个平台联动涨粉
> 定位：一个普通人的投资学习实录，不是专家教学


## 内容定位策略

```
你的独特价值 = 真实的学习过程 + 可复制的操作步骤

别人做不到的：
- 专家讲投资 → 离普通人太远，读者觉得"我做不到"
- 财经博主 → 制造焦虑，追热点，读者看个热闹

你能做到的：
- 你正在从零学，读者在看一个真实的人变懂的过程
- 你说的每一句话都是自己刚踩过的坑
- "他都可以，我也可以" → 最高的转化率
```

## 一、公众号内容规划

### 核心定位
一个普通人边学投资边记录，手把手带读者一起入门的陪伴型账号

### 栏目设计（4个固定栏目）

#### 栏目一：「投资学习日记」系列
- **频率**：每周1篇
- **内容**：本周学了什么、做了什么实操、账户盈亏情况、心得感悟
- **用户价值**：跟着你一起学，看到真实的投资过程（包括亏损）
- **参考标题**：
  - 《定投第一个月，账户变红了，我的真实反应》
  - 《本周读了这本书，一个关键认知改变了我的投资思路》
  - 《我30岁才开始理财，是不是太晚了》

#### 栏目二：「一个概念讲清楚」系列
- **频率**：每2周1篇
- **内容**：用生活化例子把一个投资概念讲透，零基础能看懂
- **用户价值**：扫盲，消除对投资术语的恐惧
- **参考标题**：
  - 《什么是复利？用一个包子铺的故事讲清楚》
  - 《PE估值到底是什么？用你熟悉的菜市场来解释》
  - 《基金名字里的A和C，这次彻底讲明白》

#### 栏目三：「手把手实操」系列
- **频率**：每2周1篇
- **内容**：具体操作教程，截图+步骤，跟着做就能完成
- **用户价值**：解决"想开始但不会操作"的核心障碍
- **参考标题**：
  - 《手把手：10分钟在支付宝开通基金定投》
  - 《如何看懂一份基金报告，关键就看这3页》
  - 《盘点我用过的5个理财App，最后只留了这2个》

#### 栏目四：「投资避坑」系列
- **频率**：每月1篇
- **内容**：自己踩过的坑、身边人的教训、经典投资错误
- **用户价值**：反常识内容容易传播，避坑比赚钱更吸引眼球
- **参考标题**：
  - 《理财第一年我犯的3个错误，每个都白扔了好几千》
  - 《为什么别人推荐的基金你买了就跌》
  - 《理财App上写着"稳健理财"，亏了我才搞懂为什么》

### 发文节奏
- 每周1-2篇（新号前期保证质量比数量重要）
- 周二/周四晚8-10点发（公众号阅读高峰）
- 每月至少1篇能引发转发的内容（避坑类或反常识类）

## 二、小红书内容规划

### 核心定位
一个在认真学投资的女生，用极简的方式讲清投资概念，记录真实成长

### 选题矩阵（4个方向 + 具体选题）

#### 方向一：极简科普卡片（拉新主力）
- **形式**：1张信息图 + 大白话文案
- **选题**：
  - 一张图看懂所有基金类型
  - 复利 vs 单利，一张表看明白
  - 定投为什么能赚钱（附微笑曲线图）
  - 余额宝 vs 债券基金 vs 指数基金，风险和收益对比
  - 你的钱该分成4笔（活钱/稳钱/长钱/保钱）
  - 指数基金到底是什么（一句话：买国运）
  - PE/市净率/股息率，第一次看基金就能用
  - 宽基指数 vs 行业指数，新手先投哪个
  - 一张表看A股/港股/美股有什么区别
  - 基金费用拆解：你亏的钱可能一半都是手续费

#### 方向二：学习日常/情绪共鸣（粘性主力）
- **形式**：真实感受分享，口语化，像在跟闺蜜聊天
- **选题**：
  - "今天第一次打开基金账户，不知道看哪里"
  - "定投第二个月亏了，但我不慌是假的"
  - "30岁才发现身边人早就在理财了"
  - "以前觉得投资要有钱才能开始，后来发现错了"
  - "这个月光存钱没花钱，居然也没觉得难受"
  - "我的10万存款配置方案（不是放余额宝）"
  - "发现自己开始主动看财经新闻的那天"
  - "说实话理财让我第一次感觉到对钱有掌控感"

#### 方向三：投资书单/资源推荐（收藏型流量）
- **形式**：多图轮播 + 每本1句核心价值
- **选题**：
  - 投资入门必读3本书（从易到难排列）
  - 理财纪录片推荐（入门不枯燥）
  - 理财App推荐（带个人使用感受）
  - 个人理财博主推荐
  - 我学理财第1个月看了什么

#### 方向四：存钱/攒钱/记账（关联引流）
- **形式**：晒图打卡 + 方法分享
- **选题**：
  - "记账3个月，我知道了自己的钱去哪儿了"
  - 我的52周存钱挑战
  - 工资到手先存后花的真实体验
  - 每个月能多存1500的几个小习惯
  - 存到10万的真实感受（比想象中平静）

### 发文节奏
- 每周3-4条（科普1条+日常1条+书单/存钱1条+互动1条）
- 晚上7-10点发（小红书活跃高峰）
- 高频测试哪种选题数据好，快速调整

```
一个概念讲清楚：复利   ←→    一张图：复利是什么（10秒看懂）
手把手：支付宝定投     ←→    定投截图打卡 + "姐妹们我开始定投了"
投资避坑：新手错误     ←→    一条避坑清单 + "你中了几条"
月度投资复盘          ←→    晒账户收益（含亏损）+ 真实感受
```

## 五、变现路径（长期）

内容涨粉是第一步，后续变现路径：

1. **广告合作**：理财App推荐、基金公司软文（粉丝量够大后）
2. **知识付费**：投资入门课程/训练营（从零开始学理财21天）
3. **社群运营**：付费理财学习社群，一起定投打卡
4. **图书出版**：内容积累到一定程度出版实体书

**注意**：不要急着变现。先在读者心中建立"这个人靠谱、说的话我能看懂、跟着做真的有用"的信任。这个信任是所有变现的基础。
